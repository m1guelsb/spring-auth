## About 🔍
In this tutorial we learn how to apply protected routes with jwt authentication and authorization in spring.

## Stack 🔧
- Java 17
- Spring-boot 3.1.5
- jwt
- hibernate/jpa
- postgresql
- lombok

---

## How to run 🏃

1 - Start the database with docker compose:
```bash
docker compose up

```
2 - Run the application
